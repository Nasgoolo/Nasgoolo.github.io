Ласкаво просимо до мого Телеграм Бота!

Опис функціоналу:

1. Показ свят, які сьогодні святкуються

Наш бот може надати вам список свят, які відзначаються сьогодні. Просто надішліть команду "holidays <код країни>" для отримання списку свят.

2. Отримання погоди для певного міста

Наш бот також надає поточну погоду для будь-якого міста. Просто надішліть команду "weather <ім'я міста>" для отримання інформації про погоду.

3. Інформація про сонячні дані для певного міста

Ви можете отримати дані про сонячний розподіл для будь-якого міста. Просто надішліть команду "sun_data <ім'я міста>" для отримання інформації про сонячні дані.

4. Інформація про курс валют

Ви можете отримати дані про курс валют. Просто надішліть команду "/exchange_rate" і ви можете дізнатися курс обміну між доларом США, євро, польським злотим та українською гривнею.

5. Додаткові функції

У нашому боті також доступні такі функції:

  • Додавання контакту: "add_contact <ім'я> <номер телефону>"
  • Отримання контакту: "get_contact <ім'я>"
  • Видалення контакту: "delete_contact <ім'я>"
  • Відображення всіх контактів: "/display_contacts"
  • Список доступних команд: "/commands"
